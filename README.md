# Home Infrastructure & Media Server Homelab

## Overview

This project documents the design, deployment, and maintenance of a self-hosted home infrastructure environment built on a Raspberry Pi.

The environment provides:

* Network-wide DNS filtering
* Media streaming services
* Secure remote access
* Reverse proxy management
* Containerized application hosting
* Home automation and monitoring

## Hardware

* Raspberry Pi
* 3TB Seagate External Drive
* Xfinity Gateway

## Core Technologies

* Raspberry Pi OS
* OpenMediaVault
* Docker
* Portainer
* Tailscale

## Services

### OpenMediaVault

Storage management and file sharing.

### Jellyfin

Media streaming server for movies and television content.

### Pi-hole

Network-wide DNS filtering and ad blocking.

### Nginx Proxy Manager

Reverse proxy management for internal services.

### Homepage

Centralized infrastructure dashboard.

### Portainer

Docker container management platform.

### Home Assistant

Home automation and infrastructure monitoring.
